功能

把日常的零碎，整理成可执行的清单

围绕「记下 · 提醒 · 整理 · 同步」四件事，Tie 提供一套互相协作的能力。

Markdown 编辑

所见即所得的 Markdown，支持图片拖入插入、待办清单、代码块、表格，所有排版都用键盘完成。

定时提醒

一次、每天、每周、每月，并支持农历每月提醒，传统节日与月历节奏一并照顾。

日历视图

月历与年历自由切换，提醒分布一目了然，点选日期即可在对应日创建便签。

标签管理

置顶、自定义颜色、透明度、归档，按标签筛选与分组，让便签按你想要的方式排列。

Git 多设备同步

数据完全本地存储，借助 Git 完成多设备同步，没有云端、没有账号，只有你掌控的仓库。

AI 分析

保存便签时自动识别提醒、拆分任务、规整文本与推荐标签，把口语化笔记变成结构化待办。

AI 文本重写

右键菜单五种操作：规整、转清单、正式、精简、温和，一行操作重塑整段表达。

FTS5 全文搜索

覆盖标题、内容、标签的秒级检索，关键词高亮命中位置，数千条便签也能精准定位。

便签模板

自定义常用模板，一键创建日报、周报、会议纪要等结构化便签，省去重复排版。

特性

三条不可妥协的底线

数据归你所有，同步由你掌控，AI 是助手而不是把关人。

01

隐私优先 · 本地存储

所有便签、提醒、标签、模板均以本地 SQLite + FTS5 索引存储，无任何云端、无任何统计 SDK，关闭网络 Tie 照常工作。

  • SQLite 持久化，FTS5 全文索引
  • 无账号、无登录、无埋点
  • 可一键导出 / 备份整库
02

Git 同步 · 你掌控的仓库

多设备同步通过你自己的 Git 仓库完成，可自建 Gitea / Forgejo / GitHub，提交历史即是便签历史，谁在什么时候改了什么一目了然。

  • 支持私有 Git 仓库，端到端由你托管
  • 自动 commit / pull / push，冲突可视
  • 无云端账户，无第三方中转
03

AI 能力 · 可插拔的助手

AI 只在你需要时介入：保存时分析结构、右键重写文本、待办智能排序、日历一键生成周报 / 月报草稿，全部可关闭、可替换模型。

  • 支持本地或远端 LLM，配置自由
  • 分析 / 重写 / 排序 / 报告四类能力
  • 未启用 AI 时核心功能完全可用
