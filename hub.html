AI 便签
v0.8.0 · 桌面便签应用
功能
· Markdown 编辑，支持图片拖入、待办清单
· 定时提醒：一次 / 每天 / 每周 / 每月 / 农历每月
· 日历视图：月历+年历，直观查看提醒分布
· 标签管理、置顶、自定义颜色、透明度、归档
· Git 多设备同步，数据完全本地存储
· AI 嗅探：保存便签自动识别提醒/拆任务/规整/标签
· AI 文本重写：右键菜单 5 种操作（规整/转清单/正式/精简/温和）
· AI 待办排序：待办 > 3 条时按紧急程度智能排序
· AI 周报/月报：日历页一键生成回顾草稿
· 多选批量操作：Ctrl+点击多选，归档/改色/删除
· 启动自动防重叠：重叠便签级联偏移 30px

使用方式
· 双击托盘图标 → 新建便签
· 输入 - [ ] 任务 → 查看模式显示可点击 checkbox
· 拖入图片 → 自动保存到本地并插入引用
· 右键便签内容 → 置顶 / 归档 / 删除 / 换色 / AI 重写
· Ctrl+S 保存 · Esc 退出编辑 · Ctrl+N 新建
· Ctrl+Shift+N 全局新建 · Ctrl+Shift+S 显示全部
· 选中文本右键 → ✨ 规整 / 📋 转清单 / 👔 正式 / 📢 精简 / 🤝 温和
· 待办 > 3 条 → 内容顶部出现 ✨ AI 排序按钮
· 日历页顶部 → 📊 生成月报 · 📋 生成周报
· Ctrl+点击便签列表 → 多选 → 底部批量操作栏
